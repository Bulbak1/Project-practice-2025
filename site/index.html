Проект EasyAccess

Расширение для браузера, которое помогает людям с ограниченными возможностями

Главная О проекте Участники Журнал Ресурсы

Аннотация проекта

Проект **EasyAccess** разрабатывается с целью улучшения доступности интернета для людей с ограниченными возможностями, в первую очередь для тех, кто сталкивается с трудностями в восприятии визуального и аудиоконтента. Это расширение для браузера предоставляет пользователям возможность адаптировать веб-страницы под их индивидуальные потребности.

С помощью **EasyAccess** каждый пользователь сможет улучшить свой опыт взаимодействия с интернет-ресурсами, независимо от физических ограничений, таких как проблемы со зрением или слухом. Это расширение предлагает простые, но эффективные инструменты для настройки контента веб-страниц, улучшая их восприятие и доступность.

Основные возможности

  • Текст в речь: Преобразование текста на странице в аудио для пользователей с нарушениями зрения.
  • Увеличение шрифта: Простое увеличение текста и изменение контраста для улучшения читаемости.
  • Автоматическая транскрипция: Транскрипция аудио- и видеоконтента для пользователей с нарушениями слуха.
  • Гибкость настроек: Пользователи могут настроить расширение под свои предпочтения для более удобного взаимодействия с веб-страницами.
