Полезные ресурсы

Здесь собраны ссылки на полезные материалы и ресурсы, которые помогут лучше понять цели и технологическую базу проекта EasyAccess, а также узнать больше о доступности в интернете.

MDN Web Docs — Доступность

Подробное руководство по созданию доступных веб-интерфейсов. Особенно полезно для разработчиков расширений.

W3C Web Accessibility Initiative

Международные стандарты и ресурсы по обеспечению цифровой доступности для людей с ограниченными возможностями.

GreasyFork — Скрипты для браузера

Каталог пользовательских скриптов, аналогичный подходу EasyAccess. Здесь можно посмотреть примеры и вдохновиться.

Web Content Accessibility Guidelines (WCAG) 2.1

Руководство по обеспечению доступности веб-контента, которое определяет, как сделать веб-контент более доступным для людей с ограниченными возможностями.

Chrome Extensions Documentation

Официальная документация по разработке расширений для браузера Chrome, которая поможет понять, как создаются и работают расширения.

GitHub - Проекты по доступности

Коллекция проектов с открытым исходным кодом на GitHub, посвященных улучшению доступности в интернете.
